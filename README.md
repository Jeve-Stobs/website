# Getting Started

### Prerequisites

- [Node.js v12.22.0](https://nodejs.org/en/) or higher

## NPM

### Install and Run Dev Server

```bash
git clone https://github.com/Jeve-Stobs/website.git
cd website

npm install
npm run dev
```

### Production Build and Start

```bash
  npm run build
  npm run start
```

## Yarn

### Install and Run Dev Server

```bash
git clone https://github.com/Jeve-Stobs/website.git
cd website

yarn
yarn dev
```

### Production Build and Start

```bash
yarn build
yarn start
```
